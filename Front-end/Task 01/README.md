## nauczcie się emmet
https://docs.emmet.io/
* `.class` // classa
* `#id` // id
* `*` // mnożnik elementów
* `+` // dodanie elementu na tym samymy poziommie
* `>` // wewnętrz elementu
* `{text}` // wyplucie tekstu
* `$` // zwraca liczbę
